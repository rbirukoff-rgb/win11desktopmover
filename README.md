# Win11DesktopMover

A lightweight utility for Windows 11 that lets you **move the active window between virtual desktops** using simple keyboard shortcuts.  
Fast, minimal, no installation required.

---

## 🚀 Features

- Move the currently focused window to the next or previous virtual desktop
- Uses Windows 11 Virtual Desktops API
- Global hotkeys that work in any application
- Single portable `.exe` with no dependencies
- Does not require administrator privileges

---

## 🎹 Keyboard Shortcuts

| Shortcut | Action |
|---------|--------|
| `Ctrl + Win + →` | Move window to the next virtual desktop |
| `Ctrl + Win + ←` | Move window to the previous virtual desktop |

*(If your actual shortcuts differ, tell me — I’ll update the table.)*

---

## 📦 Installation

1. Download the latest version from the **Releases** section  
2. Run `win11desktopmover.exe`  
3. The utility works in the background immediately

---

## 📝 Requirements

- Windows 11  
- No additional frameworks or runtimes required  
- Fully static build
